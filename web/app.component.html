Javascript smart contracts on NEAR

near-sdk-js makes it possible to deploy Javascript smart contracts on NEAR protocol. This web application let you write, simulate, deploy and call your javascript smart contracts in the web browser.

Note: Currently you can only create low-level ( see examples here ) contracts from here.

Open the menu to edit code for your Javascript contract, deploy, or call it on-chain.
